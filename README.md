## 🧭 서비스 메뉴 구조
- 👑 **관리자**
  - 사원 관리
  - 부서 / 직급 관리
- 👤 **직원**
  - 발주 / 입고 처리
  - 주문 / 출고 처리
  - 품목 관리
  - 재고 현황 조회
  - 매출매입 리포트
  - 재고 리포트
- ⚙️ **시스템**
  - 모든 사용자의 활동 로그 기록

## 🚀 핵심 기능 요약
- ✅ 상품·주문·발주·출고·입고 **풀사이클 처리**
- 📊 매출 및 재고 **리포트**로 변화 추세 파악
- 📈 활동 로그 **대시보드**로 사용 흐름 실시간 모니터링
